# 🎲 Random Task Generator

**Автор:** Ибрагимов Егор Евгеньевич

## 📌 Описание проекта

Программа для генерации случайных задач. Помогает разнообразить день и не забывать о важных делах!

### Возможности:
- 🎲 Генерация случайной задачи (учёба, спорт, работа)
- ➕ Добавление своих задач с выбором типа
- 🔍 Фильтрация истории по типу задачи
- 💾 Автоматическое сохранение истории в файл JSON
- 📂 Загрузка истории при запуске программы
- ✅ Проверка ввода (нельзя добавить пустую задачу)

## 🛠 Требования для запуска

- **Python 3.7** или новее
- Библиотеки: `tkinter`, `json`, `random`, `os` (все входят в стандартный Python)
