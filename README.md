# Random Password Generator — Генератор случайных паролей

**Автор:** Ибрагимов Егор Евгеньевич

## Описание программы

GUI-приложение для генерации надёжных случайных паролей с возможностью настройки параметров и сохранения истории.

### Функциональность:
- **Настройка длины пароля** (ползунок от 4 до 32 символов)
- **Выбор типов символов**:
  - Буквы (латиница, заглавные и строчные)
  - Цифры (0-9)
  - Специальные символы (!@#$%^&* и др.)
- **Генерация** случайного пароля с перемешиванием
- **Копирование** пароля в буфер обмена
- **История паролей** с датой и временем создания
- **Сохранение/загрузка** истории в JSON-файл
- **Очистка истории**
